Built a Ek School Web Based System for FYP.

  • Live Demo
  • Source code

Ek School is a transformative project aimed at addressing the substantial disruptions and challenges that the education sector has in Pakistan. Problems like different standards and curriculum, old teaching methodologies, or access of interactive LMS.

Key Features

  • Authentication Module
  • Admission and Registration Module
  • Admin, Teacahers, and Students Dashboard
  • LMS Module
  • Result Module
  • Attendance Module
  • Announcement Module

Technologies Used

  • Frontend: ReactJS, NextJS, Tailwand CSS
  • Backend: NodeJS, ExpressJS, MongoDB, etc.
  • Others: Git, StarUML, MS Word etc.
